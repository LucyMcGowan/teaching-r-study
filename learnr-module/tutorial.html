Vectors

Numeric vectors

R operates on data structures. The simplest data structure is a numeric vector, defined as an ordered collection of numbers. A single number is a vector of length 1, for example, in the code below, a vector with a single number, 3, is assigned to x.

x <- 3

A vector with multiple numeric components can be created using the c() function, which concatenates each element. For example, a vector consisting of three numbers, 9, 2, and 8 can be assigned to x with the following code.

x <- c(9, 2, 8)

Vector arithmetic

Arithmetic can be performed on vectors; the expressions will be applied element by element. For example, two vectors can be added using the +, as seen below.

c(1, 2, 3) + c(3, 2, 1)
## [1] 4 4 4

If the vectors are not of the same length, the elements of the shorter vector(s) will be recycled to complete the expression.

c(1, 2) + c(1, 2, 3, 4)
## [1] 2 4 4 6

For example, above the vector c(1, 2) was repeated so that it has length 4, resulting in c(1, 2, 1, 2) to match the length of c(1, 2, 3, 4).

Additional common arithmetic functions include:

  • log()
  • exp()
  • sqrt()
  • min()
  • max()
  • mean()
  • var()

Logical vectors

In addition to numeric vectors, R also allows for logical vectors. These vectors can consist of the values: TRUE, FALSE, and NA (meaning “not available” indicating a missing value). These vectors are generated by conditions, created with the following logicial operators:

  • <: less than
  • <=: less than or equal to
  • >: greater than
  • >=: greater than or equal to
  • ==: equal
  • !=: not equal
  • &: logical AND (element wise comparison)
  • &&: logical AND (evaluates left to right, examining only the first element of each vector)
  • |: logical OR (element wise comparison)
  • ||: logical OR (evalutes left to right, examining only the first element of each vector)
  • !: logical negtation (NOT)

For example, the following code creates a logical vector assessing whether each element of the vector x is greater than 4

x <- c(8, 6, 7, 5, 3, 0, 9)
x > 4
## [1]  TRUE  TRUE  TRUE  TRUE FALSE FALSE  TRUE

Character vectors

It is also possible to store character values in a vector. Character strings are entered in R using quotation marks, either ' or ", for example below is a character vector with the first three letters in the alphabet.

c("a", "b", "c")
## [1] "a" "b" "c"

Indexing vectors

You can index a vector by using a square bracket, [] with the desired index vector. There are four types of index vectors

  1. A logical vector
  2. A vector of positive integers
  3. A vector of negative integers
  4. A character vector

Logicial vectors

Using a logical vector, x, to index another vector, y, will select only the values of y for which the conditions for x are TRUE. For example, if x is c(TRUE, FALSE, TRUE), the first and third element of y will be selected. It works like this. The x vector, or the logical index vector, will be placed in a square bracket appended to the vector y.

y <- c(1, 2, 3)
y[c(TRUE, FALSE, TRUE)]
## [1] 1 3

Vectors of positive integers

Using a vector of positive integers, x, to index another vector, y, will select only the values of y that correspond to the position dictated by x. For example, if x is c(1, 2), the first and second element of y will be selected.

y <- c(3, 4, 5)
y[c(1, 2)]
## [1] 3 4

Vectors of negative integers

Using a vector of negative integers, x, to index another vector, y, will select only the values of y that correspond to the position NOT in x. For example, if x is -c(1, 2), the first and second element of y will NOT be selected.

y <- c(3, 4, 5)
y[-c(1, 2)]
## [1] 5

Vectors of character strings

If a vector has a names attribute, the elements can be selected using a character vector.

fruits <- c(1, 2, 3)
names(fruits) <- c("apple", "banana", "pear")
fruits[c("pear", "apple")]
##  pear apple 
##     3     1

Lists and Data Frames

Lists

An R list is data structure consisting of an ordered collection of components. The components can be of different modes and types, for example, a list could be a collection of different types of vectors of different lengths, as seen below.

list(numeric = c(1, 2, 3), 
     logical = TRUE,
     character = c("apple", "banana"))
## $numeric
## [1] 1 2 3
## 
## $logical
## [1] TRUE
## 
## $character
## [1] "apple"  "banana"

Components of the list are numbered and can be referenced using a double bracket. For example, you can reference the first element of a list with [[1]] as seen below.

lst <- list(numeric = c(1, 2, 3), 
     logical = TRUE,
     character = c("apple", "banana"))
lst[[1]]
## [1] 1 2 3

Components can also be referenced via their names. For example, the first element of the list below could be referenced by [["numeric"]]

lst[["numeric"]]
## [1] 1 2 3

Data frames

A data frame is a specific type of list with the class data.frame or tibble. A tibble is a specific subclass of data.frame that has a few additional restrictions. Broadly, data frames consist of rows and columns, where the length of the rows is consistent for all columns. For example, here is a data frame with two columns, x and y, each with three rows.

data.frame(
  x = c(1, 2, 3),
  y = c(4, 5, 6)
)

Manipulating data

You can manipulate data frames. In this lesson, we will learn how to:

  • Filter or subset a data frame’s rows based on particular conditions
  • Select or subset a data frame’s columns
  • Add variables to a data frame
  • Summarize columns in a data frame

Filtering or subsetting rows

To filter rows of a data frame, you can use the subset() function with the subset argument. For example, here is a data frame included in R called mtcars. This data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973–74 models).

mtcars

If you wanted to filter this data frame to only include rows of cars that have miles per gallon (mpg) greater than 25, you would do the following:

subset(mtcars, subset = mpg > 25)

You can save this new filtered data frame by assigning it to a new object using the <-:

mtcars_subset <- subset(mtcars, subset = mpg > 25)

If you want to subset based on more than one condition, you can use logical operators discussed in a previous section. For example, if you want to filter to only include rows where mpg is greater than 25 and weight (wt) is greater than 2, you would do the following:

subset(mtcars, subset = mpg > 25 & wt > 2)

Selecting or subsetting columns

To select specific columns, you can also use the subset() function. Instead of specifying the subset argument (that would filter the rows), you can specify the select argument to select specific columns. For example, if you want to create a data frame that only includes the mpg column, you can do the following:

subset(mtcars, select = mpg)

If you would like to only include both mpg and cyl, you can pass a vector, c(mpg, cyl) to the select argument.

subset(mtcars, select = c(mpg, cyl))

If you would like to include all columns except mpg, you can use the - sign as shown below.

subset(mtcars, select = - mpg)

Add variables to a data frame

There are many ways to add a column to a data frame. Two are the following:

  • via the $
  • via the cbind() function

A variable can be referenced in a data frame using the $. For example, using the mtcars data frame, you can reference the mpg column using mtcars$mpg. You can add a new variable by assigning a vector to the new variable, as below:

mtcars$new_var <- 1:32

Alternatively, a new variable can be added using the cbind() function. In order to ensure that this new variable is saved, you must assign the new data frame created with cbind()

new_var <- 1:32

mtcars <- cbind(mtcars, new_var)

Summarise columns in a data frame

In R, you can use functions to summarise observations in a column of a data frame. For example, to take the average, or mean, of a column, you can use the mean() function and reference the data frame’s column using the $. To take the average of the miles per gallon (mpg) column in the mtcars data frame, you would run the following:

mean(mtcars$mpg)
## [1] 20.09062

If you want to summarise observations in a column by another column, you can use the aggregate() function. For example, if you wanted to calculate the average miles per gallon (mpg) by the number of cylinders (cyl) using the mtcars data, you would run the following:

aggregate(mpg ~ cyl, data = mtcars, FUN = mean)

If instead of taking the average, you wanted to take sum the miles per gallon (mpg) by the number of cylinders (cyl), you would change the FUN argument to sum as below:

aggregate(mpg ~ cyl, data = mtcars, FUN = sum)
